## 概念理解

### Q: AI、机器学习、深度学习、大模型是什么关系？

**A:** 它们是包含关系：

```
AI（人工智能）          ← 最广泛的概念
└── 机器学习            ← AI的一个分支
    └── 深度学习        ← 机器学习的一种方法
        └── 大语言模型   ← 深度学习的一种应用
```

简单来说：所有大模型都是AI，但不是所有AI都是大模型。

### Q: 什么是Token？为什么很重要？

**A:** Token是大模型处理文本的基本单位。可以这样理解：

- 对人类：文字是基本单位（字、词、句）
- 对AI：token是基本单位

**为什么重要？**
1. **计费依据**: API按token数量收费
2. **容量限制**: 上下文窗口以token计量
3. **语言差异**: 中文通常1个字≈1-2个token，英文1个词≈1-2个token

### Q: 什么是"幻觉"？如何避免？

**A:** 幻觉（Hallucination）指AI生成看似合理但实际错误的内容。

**避免方法：**
1. 要求AI提供信息来源
2. 对关键信息进行人工核实
3. 使用检索增强（RAG）技术
4. 让AI在不确定时明确表示"我不知道"

## 故障排除

### Q: 安装Claude Code时提示"command not found"

**A:** 这是因为系统找不到安装的程序。解决方法：

1. 确认安装成功（重新运行安装命令）
2. 检查环境变量（书中第6章会详细讲解）
3. 重启终端后重试

### Q: API Key失效了怎么办？

**A:**
1. 登录Anthropic控制台
2. 删除旧Key，创建新Key
3. 更新配置文件中的Key
4. 重启相关程序

### Q: OpenClaw启动后无法连接

**A:** 检查以下几点：
1. API Key是否正确配置
2. 网络连接是否正常
3. 端口是否被占用
4. 查看错误日志获取详细信息
