# 零基础AI入门：从App到OpenClaw

## 课程概述

本课程面向**完全零基础的读者**：
- 无任何编程经验
- 数学只会加减乘除
- 电脑只会使用24键打开App
- 不知道什么是终端/命令行

## 学习目标

完成本课程后，学员将能够：

1. **理解层面**：理解大语言模型的核心工作原理，无需高深数学
2. **应用层面**：熟练使用Claude App、Claude Code和OpenClaw
3. **实践层面**：能够独立部署和配置个人AI助手

## 课程结构

本课程采用**理论与实践并行**的教学方式：
- 每章先介绍必要的概念（用类比而非公式）
- 紧接着是动手实践任务
- 理论完全服务于实践需求

## 核心主题

### Phase 1: 认识AI（第1-4章）
- AI到底是什么
- 如何与AI有效沟通（Prompt基础）
- AI的能力边界与局限性
- 从网页版到API：为什么需要更好的工具

### Phase 2: 进阶使用（第5-8章）
- 第一次使用终端
- 安装和配置Claude Code
- 用Claude Code处理文件和任务
- Claude Code vs Claude App：场景选择

### Phase 3: 搭建个人AI助手（第9-14章）
- OpenClaw介绍与定位
- 环境准备与安装
- 基础配置与个性化
- 技能系统与扩展
- Claude Code与OpenClaw协同使用

## 教学方法

- **类比优先**：用生活例子解释技术概念
- **台阶式爬坡**：每一步都建立在上一步基础上
- **问题导向**：理论只在解决实际问题时引入
- **错误包容**：每个关键步骤都有"如果出错了"的排查指南

## 预期成果

学员完成课程后，将拥有一个**完全自主可控的个人AI助手**，能够：
- 24x7待命响应
- 处理日常任务（搜索、文件管理、日程等）
- 与Claude Code协同完成更复杂的开发任务
