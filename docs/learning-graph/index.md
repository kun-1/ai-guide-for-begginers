# 学习图谱

本教材采用**学习图谱（Learning Graph）**组织知识结构，包含180+核心概念，通过可视化方式展示概念之间的依赖关系。

## 什么是学习图谱？

学习图谱是一种知识组织方式，它将课程中的每个概念视为一个**节点**，概念之间的**依赖关系**视为连接节点的边。

### 核心特性

- **前置依赖**：学习某个概念前，必须先掌握哪些概念
- **层次结构**：从基础概念到高级概念的清晰路径
- **可视化探索**：交互式图谱，自由浏览知识网络

## 如何使用学习图谱

### 对于学习者

1. **探索模式**：浏览图谱，发现概念之间的关联
2. **学习路径**：查看任意概念的前置知识要求
3. **查漏补缺**：检查自己尚未掌握的基础概念

### 对于教师

1. **课程设计**：确保教学顺序符合概念依赖关系
2. **难度控制**：了解每个概念的前置要求复杂度
3. **个性化推荐**：根据学生已掌握概念推荐学习路径
